Research & Engineering Projects

This page highlights selected projects demonstrating experience in data engineering, machine learning, trustworthy and explainable AI, and scalable intelligent systems, with emphasis on robust data pipelines, graph-based learning, and AI-driven decision support.

Selected Projects

  1. Scalable Data Engineering Pipeline for AI-Driven Analytics
    Designed and implemented an end-to-end data engineering pipeline supporting large-scale data ingestion, cleaning, transformation, and storage for downstream machine learning tasks. The system emphasizes data quality validation, schema evolution handling, and reproducible feature engineering for AI applications.

  2. Graph-Based Knowledge Representation and Reasoning System
    Developed a graph-centric data model to represent complex relational information and enable efficient reasoning over interconnected entities. The project supports graph querying, graph-based inference, and integration with machine learning models for intelligent decision-making.

  3. Trustworthy Machine Learning Framework with Explainability
    Implemented a machine learning framework incorporating model interpretability, fairness-aware evaluation, and robustness analysis. The system provides post-hoc explanations and diagnostic insights to support transparent and accountable AI deployment.

  4. Real-Time Intelligent Monitoring and Alerting System
    Built a real-time data processing and analytics platform capable of ingesting high-velocity data streams and generating timely alerts using predictive models. The system prioritizes low-latency processing, reliability, and fault tolerance for mission-critical environments.

  5. Simulation and Modeling of Complex Adaptive Systems
    Developed simulation models to analyze the behavior of complex systems under dynamic and uncertain conditions. The project enables scenario analysis, stress testing, and evaluation of system-level robustness for data-driven policy and planning.

  6. Advanced Database System for Structured and Graph Data
    Designed an advanced data storage and query architecture supporting both relational and graph-based data. Focused on query optimization, indexing strategies, and scalable data access patterns required by modern AI workloads.

  7. Secure Data Processing and AI Pipeline
    Implemented security-aware data pipelines integrating access control, data integrity verification, and secure model deployment practices. The project addresses risks in data handling and AI workflows, supporting secure and trustworthy intelligent systems.

  8. AI-Driven Decision Support System
    Developed an intelligent decision support platform combining machine learning models, data analytics, and visualization tools to assist strategic and operational decision-making. The system emphasizes reliability, interpretability, and alignment with organizational and institutional priorities.
